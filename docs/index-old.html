Launch Sequence

~58s setup

spectral-cortex ingest --repo . --out smg.json --rebuild
spectral-cortex query --query "mcp" --smg smg.json --json

Auto-pilot loop: spectral-cortex update --repo . --out smg.json --rebuild --git-filter-preset git-noise

Built for real agent workflows, not demo retrieval.

Most memory systems summarize with LLMs and lose fidelity. Spectral Cortex keeps structure in math: embeddings, graph Laplacian, spectral neighborhoods, and explicit links between semantically distant but topologically connected changes.

What you get

Semantic + temporal ranking

Retrieve by meaning, then re-rank by recency with configurable temporal weight and half-life.

Incremental Git ingestion

Use update in post-commit hooks. Only unseen commits are embedded, then rebuild if needed.

Long-range links

Expose spectrally related notes with similarity scores for cross-topic debugging and context jumps.

Command-line first

Production-friendly single binary: ingest, update, query, note inspection, MCP server.

  • ingest for initial graph build
  • update for incremental maintenance
  • query for ranked retrieval
  • note for per-note related-link inspection
  • mcp for stdio MCP tool serving

MCP server, token-efficient by design

Responses are compact markdown (not bloated JSON dumps).
